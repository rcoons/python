## python面试基本常识

### 1.Python解释器种类以及特点

CPython

****c语言开发的 使用最广的解释器****

IPython

****基于cpython之上的一个交互式计时器 交互方式增强 功能和cpython一样****

PyPy

****目标是执行效率 采用JIT技术 对python代码进行动态编译，提高执行效率****

JPython

****运行在Java上的解释器 直接把python代码编译成Java字节码执行****

IronPython

****运行在微软 .NET 平台上的解释器，把python编译成. NET 的字节码****
